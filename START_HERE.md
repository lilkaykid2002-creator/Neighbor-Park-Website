## 🛠️ 调试技巧

### 打开浏览器控制台
```
Windows: F12 或 Ctrl+Shift+I
Mac: Cmd+Option+I
```

### 查看购物车数据
```javascript
// 控制台输入
cart.getCart()
```

### 查看订单数据
```javascript
// 控制台输入
JSON.parse(localStorage.getItem('neighborpark_orders'))
```

### 清空购物车
```javascript
// 控制台输入
cart.clearCart()
```

## 🎨 界面特点

### 圆点轮播
- **位置**：Hero区域底部中央
- **样式**：白色圆点，半透明
- **激活**：完全不透明，放大1.2倍
- **交互**：点击切换，停留5秒

### 购物车徽章
- **位置**：导航栏Cart旁边
- **样式**：黑色圆形，白色数字
- **更新**：实时更新商品数量
- **隐藏**：数量为0时自动隐藏

### 通知提示
- **位置**：右上角
- **样式**：黑色背景，白色文字
- **动画**：滑入效果
- **时长**：3秒自动消失

## ⚠️ 重要提示

### localStorage存储
```
所有数据存储在浏览器本地：
- 购物车数据
- 订单数据
- 商品数据

注意：
- 清除浏览器缓存会丢失数据
- 隐私模式不会保存数据
- 不同浏览器数据不共享
```

### 模拟支付
```
这是模拟支付系统：
- 二维码仅为演示
- 不会真实扣款
- 订单数据本地保存
```
