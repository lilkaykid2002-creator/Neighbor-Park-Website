## 🚀 使用说明

### 快速开始
```
1. 下载所有文件到同一文件夹
2. 双击 index.html
3. 开始使用
```

```
- 点击圆点切换幻灯片
- 点击后停留5秒
- 无操作时3秒自动播放
```

### 添加纪录片
```
1. Admin → Films → + Add New Film
2. 填写标题、分类、摘要、日期
3. 图片（可选）：
   - URL链接：粘贴图片地址
   - 本地上传：点击或拖拽
4. 视频（可选）：
   - URL链接：YouTube/Bilibili等
   - 本地上传：点击或拖拽视频
5. 保存
```

**推荐：URL链接**
```

✅ 优点：不占localStorage空间

**可选：本地文件**
```

## 🔧 故障排除

### Q: 保存成功但主页不显示？

**检查：**
1. 打开浏览器控制台（F12）
2. 查看Console选项卡
3. 刷新主页
4. 检查是否有错误信息

**解决：**
- 清除浏览器缓存
- 确保所有文件在同一目录
- 检查localStorage是否已满

### Q: 视频无法播放？

**原因：**
- 视频URL不正确
- 视频是私密的
- 格式不支持

**解决：**
1. 确认视频URL可以在浏览器打开
2. 使用YouTube测试视频
3. 检查控制台错误信息
4. Base64视频请确保文件不太大

### Q: 拖拽上传不工作？

**检查：**
1. 切换到"本地上传"标签
2. 拖到正确的上传区域
3. 确保文件格式正确

### Q: localStorage已满？

**解决：**
1. 删除不需要的内容
2. 优先使用URL链接
3. 清除浏览器数据：
   ```
   设置 → 隐私 → 清除浏览数据
   → 勾选"网站数据"
   ```
