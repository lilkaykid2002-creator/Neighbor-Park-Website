### 添加内容
```

2. + Add New
3. 填写信息
4. 上传图片/视频

5. Save
```

## ⚠️ 注意事项

### localStorage限制
- 浏览器限制：5-10MB
- 大文件建议用URL
- 清除数据会丢失
- 建议定期备份

### 推荐做法
- ✅ 优先使用URL
- ✅ 在线图床
- ✅ 视频平台
- ⚠️ 本地文件注意大小
